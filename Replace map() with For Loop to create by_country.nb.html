Replace map() with For Loop to create by_country

Introduce

In textbook (Authors:Garrett Grolemund and Hadley Wickham ,https://r4ds.had.co.nz/), there is a data frame named by_country(refer to 25.2.2 List-columns). Authors used function map() to create the data frame which includes column model presenting lifeExp~year models for each country.

Let’s find a way to replace map with for loop to create by_country.

code

by_country <- gapminder %>% 
  group_by(country, continent) %>% 
  nest()
by_country$data[[1]]
country_model <- function(df) {
  lm(lifeExp ~ year, data = df)
}

# this is original code in textbook:models <- map(by_country$data, country_model)
models <- function(df) {
  out <- vector("list",length(df))
  for (i in seq_along(df)) {
    out[[i]] <- country_model(df[[i]])
  }
  out
}
by_country <- by_country %>% 
  mutate(model = models(data))
